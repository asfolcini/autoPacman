# autoPacman

A simple, autonomous Pac-Man game built in HTML5 and JavaScript.

## Features

- **Autonomous Movement**: Pac-Man moves continuously, changing direction randomly when blocked by walls.
- **Ghost AI**: Four colored ghosts move randomly through the maze, avoiding walls.
- **Score System**: Collect dots to increase your score.
- **Lives System**: Start with 3 lives; lose one on contact with a ghost. Game over at 0 lives.
- **Game Reset**: After game over, the page automatically reloads after 5 seconds.
- **Visual Design**: Classic Pac-Man style with eyes and pupils on ghosts, yellow pacman, and white dots.
- **No Teleportation**: Walls block movement — no screen wrapping.

## How to Play

1. Open `index.html` in any modern browser.
2. Pac-Man moves automatically — no player input needed.
3. Watch as he navigates the maze, collects dots, and avoids ghosts.
4. If caught by a ghost, you lose a life. Game resets after 5 seconds.

## Technical Details

- Built with vanilla JavaScript and HTML5 Canvas.
- Maze defined as a 20x20 grid.
- Movement is grid-based, with smooth animation.
- Collision detection uses Euclidean distance.
- No external libraries or frameworks.
